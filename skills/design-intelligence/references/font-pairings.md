# Font Pairings

Max 2 families. Weights are hierarchy, not decoration. Always name the fallback stack.

## Fraunces + Inter (serif+sans - editorial, premium brand)
Literary warmth meets clean function. display: Fraunces (Fontsource), 600/900. body: Inter (Fontsource), 400/500.
h1 56px/900, h2 36px/600, body 17px/400, caption 13px/500. Fallback: Fraunces, Georgia, serif / Inter, system-ui, sans-serif.

## Lora + Source Sans 3 (serif+sans - publishing, long-form docs)
Readable serif, quiet workhorse sans. display: Lora (Google Fonts), 600/700. body: Source Sans 3 (Fontsource), 400/600.
h1 40px/700, h2 28px/600, body 18px/400, caption 14px/600. Fallback: Lora, Georgia, serif / "Source Sans 3", Arial, sans-serif.

## Space Grotesk + Inter (sans+sans - tech products, dashboards)
Geometric edge, neutral clarity below. display: Space Grotesk (Fontsource), 500/700. body: Inter (Fontsource), 400/500.
h1 48px/700, h2 30px/500, body 16px/400, caption 12px/500. Fallback: "Space Grotesk", system-ui, sans-serif / Inter, system-ui, sans-serif.

## General Sans + Inter (sans+sans - startups, product marketing)
Friendly confidence, unfussy body text. display: General Sans (Fontsource), 600/700. body: Inter (Fontsource), 400/500.
h1 52px/700, h2 32px/600, body 17px/400, caption 13px/500. Fallback: "General Sans", system-ui, sans-serif / Inter, system-ui, sans-serif.

## JetBrains Mono + Inter (mono-accent - devtools, technical docs)
Code-native accents, plain-language body. display: JetBrains Mono (Fontsource), 500/700, used only for labels/code. body: Inter (Fontsource), 400/500.
h1 44px/700 (Inter), code 14px/500 (JetBrains Mono), body 16px/400, caption 12px/500. Fallback: "JetBrains Mono", Consolas, monospace / Inter, system-ui, sans-serif.

## IBM Plex Mono + IBM Plex Sans (mono-accent - fintech terminals, data tools)
Matched family, mono for numbers and tags. display: IBM Plex Sans (Fontsource), 600/700. accent: IBM Plex Mono (Fontsource), 500, for figures.
h1 40px/700, numerals 15px/500 (mono), body 15px/400, caption 12px/500. Fallback: "IBM Plex Sans", system-ui, sans-serif / "IBM Plex Mono", Consolas, monospace.

## Bricolage Grotesque + Public Sans (display+humanist - bold consumer brands)
Playful display, grounded humanist body. display: Bricolage Grotesque (Fontsource), 700/800. body: Public Sans (Fontsource), 400/500.
h1 64px/800, h2 34px/700, body 17px/400, caption 13px/500. Fallback: "Bricolage Grotesque", system-ui, sans-serif / "Public Sans", Arial, sans-serif.

## Clash Display + Satoshi (display+humanist - agencies, portfolios)
Confident geometric headline, warm humanist body. display: Clash Display (Fontshare), 600/700. body: Satoshi (Fontshare), 400/500.
h1 60px/700, h2 32px/600, body 16px/400, caption 12px/500. Fallback: "Clash Display", system-ui, sans-serif / Satoshi, system-ui, sans-serif.

## Inter Variable (variable-font single-family - lean product UI)
One family, weight axis carries all hierarchy. font: Inter Variable (Fontsource), weight axis 400-800.
h1 40px/800, h2 24px/700, body 16px/400, caption 13px/500. Fallback: "Inter Variable", Inter, system-ui, sans-serif.

## Recursive (variable-font single-family - adaptive interfaces)
Single axis-driven family, mono/sans/casual in one file. font: Recursive (Fontsource), MONO and CASL axes plus weight 400-900.
h1 44px/800, h2 26px/700, body 16px/400, code 14px/500 (MONO axis 1). Fallback: Recursive, "JetBrains Mono", system-ui, sans-serif.
